The quickest way to get start Apache Cordova Hybrid Application (Browser + Android + iOS + Windows), just clone the project:

```bash
$ git clone https://github.com/arjunkhetia/Cordova-Project.git
```

For a complete list of platforms you can add, run:

```bash
$ cordova platform
```

You need to add a platform for which you want to build your app:

```bash
$ cordova platform add browser
```

To execute the application run:

```bash
$ cordova run browser
```
